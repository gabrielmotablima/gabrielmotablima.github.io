Churn solutions for telecom client

  • Category: Data Science & Data Analysis
  • Objective: Propose strategies to reduce churn
  • Project URL: Churn Solutions for Telecom Client

Description

✨ The current project is a basic approach of data analysis and machine learning modeling to get business insights and propose strategies of customer segmentation and churn prediction to mitigate the high churn rate.

✨ Data analysis highlighted characteristics that directly impact financial return and customer loss.

✨ Customer segmentation highlighted important profiles for the application of targeted retention and revenue strategies.

✨ With the developed predictor, we were able to mitigate user loss by up to 81%.

Made with: Python, Scikit-Learn, Pandas, NumPy, Matplotlib, and Seaborn.
